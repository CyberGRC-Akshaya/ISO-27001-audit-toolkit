## 📊 Audit Methodology

This toolkit follows **ISO 19011:2018** (Guidelines for Auditing Management Systems):

```
1. AUDIT INITIATION
   └── Define scope, criteria, objectives
   
2. DOCUMENT REVIEW
   └── Policies, procedures, risk register, previous audit results
   
3. AUDIT PLANNING
   └── Audit plan, team assignments, schedule
   
4. AUDIT EXECUTION
   ├── Opening meeting
   ├── Evidence gathering (interviews, observation, document review)
   └── Audit findings documentation
   
5. AUDIT REPORTING
   ├── Nonconformities classified (Major/Minor/Observation)
   └── Audit report issued within 5 business days
   
6. FOLLOW-UP
   └── CAR tracking, evidence of correction, closure
```

## 🚀 Quick Start

**For your first audit cycle:**
1. Download `templates/audit-plan-template.md` — adapt to your scope
2. Use `checklists/annex-a-master-checklist.md` during fieldwork
3. Log findings in `templates/NC-observation-log.md`
4. Issue report using `templates/audit-report-template.md`
5. Run `scripts/risk_heatmap.py` to visualise your compliance posture
